You need to fetch some libraries to build the goodies.
Drop the .dll and (for IKVM) .exe files in this dir.

Base libraries:
* ICSharpCode.SharpZipLib.dll
  http://www.icsharpcode.net/OpenSource/SharpZipLib/
* log4net.dll
  http://logging.apache.org/log4net/
* Lucene.Net.dll
  http://incubator.apache.org/lucene.net/
* Nini.dll
  http://nini.sourceforge.net/

Used for the updater daemon:
* CookComputing.XmlRpc.dll
  http://www.xml-rpc.net/

Used for the bulk updater:
* MWDumper
  http://download.wikimedia.org/tools/
* IKVM
  http://www.ikvm.net/
