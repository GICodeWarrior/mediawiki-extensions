'''Mikołajki''' is een [[Lijst van Poolse steden|stad]] in het [[Polen|Poolse]] [[woiwodschap]] [[Ermland-Mazurië]], gelegen in de [[powiat]] [[Powiat Mrągowski|Mrągowski]]. De oppervlakte bedraagt 8,85 km², het inwonertal 3849 (2005).
