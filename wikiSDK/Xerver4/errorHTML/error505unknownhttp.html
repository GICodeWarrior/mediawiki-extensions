Error: 505 HTTP Version not supported



It seems like you are using a very unusual HTTP-version which is not supported by Xerver. Xerver supports all existing HTTP versions by today (2003), which means HTTP/0.9, HTTP/1.0 and HTTP/1.1.
Please upgrade your browser to Netscape 1.0, Internet Explorer 1.0 (or later) or another browser of your choice supporting HTTP/0.9, HTTP/1.0 or HTTP/1.1.
