Go to Xerver Web Setup


What is this?

All Xerver settings are configurable through a Web Interface.
There are two different pages where you can change settings.
One for the HTTP server and another one for the FTP server.

Xerver Web (HTTP) Setup
You can change settings to the HTTP server here: http://127.0.0.1:32123/
Xerver FTP Setup
You can change settings to the FTP server here: http://127.0.0.1:32124/


When Xerver is not running, the setup websites are
not running and the websites will not be accessible.
