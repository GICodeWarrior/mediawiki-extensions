Congratulations!
This is the last step of this setup.
You have to decide which user interface you want to use when you run Xerver.
Most people should choose the recommended default layout.

Xerver Default - All features enabled (Recommended)!
  Most users should choose this. Everything is activated by default.

Xerver Default - Starting minimized
  This is the same as the default option, but the window is started minimized. This means for example that no Xerver-tab will be shown in the Windows taskbar.

Xerver Minimum
  This will run just the core Xerver webserver kernel and nothing else. There is no user interface activated, so you will have to kill the process using CTRL+ALT+DEL (or similar on your operating system).
The Setup pages are also disabled and you will have to manually start Setup.exe (or "java Setup") to run Setup.
There is no way to start Xerver FTP server using Xerver Minimum.

Xerver Minimum - With GUI
  This is like "Xerver Minimum", but there is a trivial user interface allowing you to close the Window to shut down Xerver. You don't have to kill the process with CTRL+ALT+DEL when you want to terminate the application.


What shall I choose?
If you are unsure what to pick, you probably should choose Xerver Maximum which is the recommended options for most users.
