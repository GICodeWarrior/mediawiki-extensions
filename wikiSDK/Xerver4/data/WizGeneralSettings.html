Welcome to Xerver Setup!

This setup consists of 9 quick steps.
You can use the menu to the left to jump to another step in this setup.
However, changes you make will only be saved when you press the Next-button (below).

By telling the Xerver auto configuration tool for what purpose are you going to use the server Xerver will automatically be configured to best suit your needs.

You can of course change settings again during the setup if you are not satisfied with the default configuration.


I will use Xerver as a public webserver, allowing everyone to view my homepage(s).

I want to use Xerver to test my CGI-scripts on my local computer. No one but myself should be able to read the content on my hard drive.

I want to use Xerver as a file-sharing server. Only people with a username and a password should have access to the files I have shared.

I have run setup before! Don't run auto config!

Note: Please choose the option which best describes your needs. After this step Xerver will be configured and you will be taken to Xerver Setup, where you can change settings to Xerver.
