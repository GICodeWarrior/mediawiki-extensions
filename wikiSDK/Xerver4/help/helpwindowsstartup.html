How do I start Xerver when Windows starts?

You can choose to run Xerver as a hidden process when you start your computer. This can be good if you want run Xerver as a server or if you just want Xerver to always be running, but you don't want see the Xerver window.

If you want to auto start Xerver Web Server as a background process at Windows startup:

  1. Copy XerverMinimumResources.exe from the folder otherStartupFiles to the Xerver directory (where xerver.jar is located).
  2. Create a shortcut to XerverMinimumResources.exe. The shortcut should be located in the startup folder. The startup folder is (depending on what version of Windows you use) located at
    Start > Programs > Startup
    or
    Start > Programs > Autostart.

If you want to auto start Xerver FTP Server as a background process at Windows startup:

  1. Copy FTPServer.exe from the folder otherStartupFiles to the Xerver directory (where xerver.jar is located).
  2. Create a shortcut to FTPServer.exe. The shortcut should be located in the startup folder. The startup folder is (depending on what version of Windows you use) located at
    Start > Programs > Startup
    or
    Start > Programs > Autostart.

Note that Xerver will start and run without a window (GUI). If you want to close Xerver, just press CTRL + ALT + DEL and delete "javaw.exe". To stop Xerver from starting at Windows startup in the future, just delete the shortcuts you just created in the autostart folder.
