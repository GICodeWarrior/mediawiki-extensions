Password protection to folders

Xerver is supporting password protection to folders.
When a user visits a password protected folder, the user will be prompted for a username and password, which you have created during the setup.
When the user enters a valid username and password the user will be able to view the content in that particular folder (and subfolders if you choosed to protect subfolders during setup). However, all folders that you password protect must belong to a "zone". The user will (with most modern browsers) see the name of the "zone" when he is asked to enter his username and password. With most modern browsers, the user will not be prompted to reenter the username and password when traveling between different folders as long as the folders are protected with the same username/password.
