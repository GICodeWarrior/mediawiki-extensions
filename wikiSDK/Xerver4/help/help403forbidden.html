403 Forbidden

A "403 Forbidden" error can be created for many reasons.
With each 403 error you get an explaination why you have received the 403 error.
Usually the reason why you have got a 403 error is obvious.
However, if you have got a 403 error and you don't know for what reason, please use this list. It can help you find the reason why you have received an unwanted 403 error:

  • ERROR MESSAGE: You have tried to reach a folder you do not have permission to access.
    You have probably not shared the folder you try to view.

    SOLUTION 1:
    Some users forget to share their root. If your root is c:\homepage\, make sure to share that folder as well! For security reasons, the root is not shared by automatic. Use the setup to share the folder you have set as root.

    SOLUTION 2:
    Some users forget to share their aliases. If you have an alias pointing to c:\homepage\, make sure to share that folder as well! For security reasons, aliases are not shared by automatic. Use the setup to share the folders you are pointing to with aliases.

  • ERROR MESSAGE: You have tried to download a file with an extension that you are not allowed to download.
    You have probably not shared the file extension your index file is using, for example html or htm.

    SOLUTION: Share all file extensions you have used in your list of index files. Example: If your index files are index.html and index.htm, make sure you have allowed users to download files with html and htm extensions.
