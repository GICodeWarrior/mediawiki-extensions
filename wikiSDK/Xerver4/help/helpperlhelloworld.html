Perl Hello World-script

You can test Xerver with this Hello World-script:
Note that if the script does not work, try to change #!/usr/bin/perl to the location of your perl interpreter (for example d:\perl\bin\perl.exe).

Save this file as "testscript.pl":

#!/usr/bin/perl
print "Content-Type: text/html \r\n\r\n";
print "Hello World!";
print "<BR>";
print "This is my first script ever!";
