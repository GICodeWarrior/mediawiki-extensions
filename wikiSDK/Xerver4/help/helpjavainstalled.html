How do I know if I have Java installed?

If you are using Windows you can check for an Java icon in your control panel. If you find an icon labeled "Java Plug-in" you have Java installed. If you can't find this icon, you must install Java on your computer to be able to run Xerver.
Java can be downloaded from:
http://java.sun.com/j2se/downloads.html

It is strongly recommended to always run the latest version of Java!

If you are unsure about which link you shall download from, go to the download site and choose JRE and "Windows (all languages, including English)".
