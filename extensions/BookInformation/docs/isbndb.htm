ISBNdb.com for BookInformation

The ISBNdb.com book information driver allows the BookInformation extension to use ISBNdb.com as a provider for ISBN lookup data.

Prerequisites

You will need an ISBNdb.com account, and you will need to have created an access key.

Configuration

$wgBookInformationProvider = 'IsbnDb';
$wgBookInformationService = array(
	'accesskey' => {{ACCESSKEY}},
);

where {{ACCESSKEY}} is your access key. You can create an access key through the ISBNdb.com developer centre.

Available Information

  • Title
  • Author
  • Publisher

Other

  • The data provider will be listed as "ISBNdb.com"
