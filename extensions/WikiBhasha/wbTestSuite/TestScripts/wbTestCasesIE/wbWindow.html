BVT_TUNG200 [Documentation] TestCase - TUNG200 - Maximize and Minimize button  
[Tags] BVT
  Log Summary :- Maximize and Minimize button    
  Log Description :- Verifying the "square button" which is on the top bar    
  Open Wikipedia ${wbLang} ${SourceSearchTextExist}  
  Execute_script ${BookMarklet}    
  Wait Until page contains element ${HistoryContainerID} ${DELAY}  
  Page should Contain Element ${MaximizeBtnID}    
  ${Status}= Is Element Present xpath=//a[@id='${MaximizeBtnID}' and @title='${MaximizeBtnToolTip}']  
  Should be true ${Status} Tool tip of Maximize button is not ${MaximizeBtnToolTip}  
  ${Height}= Get Attribute ${wbBgContentAreaXapath}@OffsetHeight  
  ${Width}= Get Attribute ${wbBgContentAreaXapath}@OffsetWidth  
  Click Element ${MaximizeBtnID} Dont Wait  
  ${Status}= Is Element Present xpath=//a[@id='${MaximizeBtnID}' and @title='${MinimizeBtnToolTip}']  
  Should be true ${Status} Tool tip of Maximize button is not ${MinimizeBtnToolTip}  
  ${NewHeight}= Get Attribute ${wbBgContentAreaXapath}@OffsetHeight  
  ${NewWidth}= Get Attribute ${wbBgContentAreaXapath}@OffsetWidth  
  ${Status}= Evaluate int(${NewHeight}) >  int(${Height})  
  Should be true ${Status} Height of the two pane screen did not increased on maximizing  
  ${Status}= Evaluate int(${NewWidth}) >  int(${Width})  
  Should be true ${Status} Width of the two pane screen did not increased on maximizing  
  Click Element ${MaximizeBtnID} Dont Wait  
  Test Pass
