This folder contains test suite for WikiBhasha extension.

The suite uses selenium functional testing application combined with robot framework.

The testcase.xls lists 42 test cases for WikiBhasha. Out of these 42 test cases 33 are succeeding and rest of the test cases need to be written correctly to succeed with WikiBhasha.

The test suite is compatible with IE and Firefox both but the results shown on the testcases.xls are using IE and French language as target language.

To run the test suite for other target language please refer the installation document.
